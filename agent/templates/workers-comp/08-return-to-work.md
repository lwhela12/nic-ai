## VIII. Return to Work / Light Duty

### NRS 616C.475 — Modified Duty Requirements
- Employer must offer modified duty if available and within employee's restrictions
- Modified duty must be meaningful and consistent with medical restrictions
- Employee must accept reasonable modified duty or risk TTD benefit suspension
- Employer cannot create "make work" positions solely to terminate benefits

### Light Duty Assessment
- **Legitimate modified duty:** Reduced hours, restricted lifting, seated work, administrative tasks within normal operations
- **Questionable modified duty:** Significantly different job with no relation to original position, humiliating assignments, positions created solely to cut benefits
- **Document:** All offered positions, restrictions, and employee's response

### Return to Work Process
1. ATP provides work status report with specific restrictions
2. Employer evaluates available positions matching restrictions
3. Employer makes written offer of modified duty
4. Employee accepts or provides written reason for refusal
5. If accepted, monitor compliance with restrictions
6. If refused, insurer may suspend TTD (with proper notice)

### Employee Rights During Modified Duty
- Cannot be required to work beyond medical restrictions
- Entitled to continued medical treatment
- May refuse if offered position doesn't match restrictions
- Protected from retaliation for filing WC claim (NRS 616B.012)
- Wage differential benefits if modified duty pays less

### Common Issues
- Employer doesn't honor restrictions (e.g., requires lifting beyond limit)
- Modified duty assignment is retaliatory or punitive
- Employee is terminated shortly after returning to modified duty
- Employer claims no modified duty available to force resignation
- Restrictions change — need updated work status reports

### Documentation Requirements
- Keep copies of all work status reports
- Document any violations of restrictions
- Record all communications about modified duty offers
- Track wages during modified duty period vs. pre-injury wages
- Report any worsening of condition during modified duty
